gdb
-----
[linux tools](http://linuxtools-rst.readthedocs.io/zh_CN/latest/tool/gdb.html)  
[cgdb](https://cgdb.github.io/)

gdb + programName 进入gdb调试模式，q退出。  
l : 输出代码  
break + 行数 ： 在第几行设置断点  
r ： run 运行程序  
c  : continue 命令  
n ：单步运行  
p ： print 达因变量  
bt ：查找函数堆栈  
q ： 退出gdb  
静态程序检测工具  
动态程序检测工具  
程序调试方法：gdb, printf, core dump, log
